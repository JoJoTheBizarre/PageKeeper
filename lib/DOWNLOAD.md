# Required Libraries

You need to download these libraries manually:

## 1. Turndown.js (HTML to Markdown converter)
- Download from: https://unpkg.com/turndown@7.1.2/dist/turndown.js
- Save as: `lib/turndown.min.js`

## 2. jsPDF (PDF generation)
- Download from: https://cdnjs.cloudflare.com/ajax/libs/jspdf/2.5.1/jspdf.umd.min.js
- Save as: `lib/jspdf.min.js`

## Quick Download Commands

If you have curl installed:

```bash
cd lib

# Download Turndown
curl -o turndown.min.js https://unpkg.com/turndown@7.1.2/dist/turndown.js

# Download jsPDF
curl -o jspdf.min.js https://cdnjs.cloudflare.com/ajax/libs/jspdf/2.5.1/jspdf.umd.min.js
```

Or visit the URLs in your browser and save the files.
